
# Day 4 - To-Do App

A simple **command-line To-Do application** in Python to manage daily tasks.  
You can **add, view, update, and delete tasks** with a simple menu interface.

## Features

- Add tasks with a **name** and **status** (Start, In Progress, Done)
- View all tasks
- Update tasks (mark as done or edit task name)
- Delete tasks
- Error handling for invalid inputs
